Cú pháp CSS cơ bản

CSS (Cascading Style Sheets) được sử dụng để định dạng và bố cục các phần tử HTML.

Cấu trúc một khai báo CSS

selector {
  property: value;
  property: value;
}

Ví dụ:

body {
  font-family: Arial, sans-serif;
  background-color: #f0f0f0;
  margin: 0;
  padding: 20px;
}

.header {
  color: #333;
  text-align: center;
  padding: 20px;
}

Box Model

Mỗi phần tử HTML đều là một hộp (box) với các thành phần: content, padding, border, margin.

Nội dung

Content → Padding → Border → Margin

.box {
  width: 200px; /* Chiều rộng nội dung */
  height: 100px; /* Chiều cao nội dung */
  padding: 20px; /* Khoảng cách giữa nội dung và viền */
  border: 5px solid #333; /* Viền */
  margin: 30px; /* Khoảng cách bên ngoài viền */
  box-sizing: border-box; /* Quan trọng: bao gồm padding và border trong width/height */
}

Flexbox

Flexbox giúp sắp xếp các phần tử một cách linh hoạt và responsive.

1
2
3
.container {
  display: flex;
  justify-content: center; /* Căn ngang */
  align-items: center; /* Căn dọc */
  flex-wrap: wrap; /* Cho phép xuống dòng */
  gap: 10px; /* Khoảng cách giữa các mục */
}

.item {
  flex: 1; /* Chia đều không gian */
  min-width: 100px; /* Độ rộng tối thiểu */
}

CSS Grid

CSS Grid cho phép tạo bố cục hai chiều (hàng và cột) một cách mạnh mẽ.

.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 cùng bằng nhau */
  grid-template-rows: 100px 200px; /* 2 hàng với chiều cao khác nhau */
  gap: 15px; /* Khoảng cách giữa các ô */
  grid-template-areas:
    "header header header"
    "sidebar content content";
}

.item {
  grid-area: header; /* Chỉ định vùng */
}

Responsive Design

Thiết kế responsive đảm bảo trang web hiển thị tốt trên mọi kích thước màn hình.

Media Queries

/* Mobile First */
.container {
  padding: 10px;
}

/* Tablet */
@media (min-width: 768px) {
  .container {
    padding: 20px;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .container {
    max-width: 1200px;
    margin: 0 auto;
  }
}

Đơn vị responsive

  • %: Tỷ lệ phần trăm so với phần tử cha
  • vw/vh: Tỷ lệ phần trăm so với viewport (khung nhìn)
  • rem: Tỷ lệ so với kích thước font gốc (thường là 16px)
  • em: Tỷ lệ so với kích thước font của phần tử cha

CSS Animations & Transitions

Tạo hiệu ứng chuyển động và chuyển tiếp giúp trang web sinh động hơn.

Hiệu ứng!
/* Transition */
.button {
  transition: all 0.3s ease;
  background-color: blue;
}
.button:hover {
  background-color: darkblue;
  transform: translateY(-3px);
}

/* Animation */
@keyframes slide-in {
  from { transform: translateX(-100%); }
  to { transform: translateX(0); }
}

.element {
  animation: slide-in 0.5s ease forwards;
}

Lời khuyên khi học CSS

  • Học và thực hành từng khái niệm một
  • Sử dụng Developer Tools (F12) để kiểm tra và chỉnh sửa CSS
  • Xây dựng các dự án nhỏ để áp dụng kiến thức
  • Tham khảo các trang như MDN Web Docs, CSS-Tricks
  • Khi đã nắm vững CSS cơ bản, chuyển sang học Tailwind CSS sẽ dễ dàng hơn

Hãy bắt đầu với những kiến thức cơ bản nhất, thực hành thường xuyên, và dần dần bạn sẽ thành thạo CSS!
